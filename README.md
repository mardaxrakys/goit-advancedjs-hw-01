# 📸 Галерея зображень та Форма зворотного зв'язку

Цей проєкт містить дві основні функції:
1. **Галерея зображень** – використання бібліотеки `SimpleLightbox` для перегляду зображень у модальному вікні.
2. **Форма зворотного зв'язку** – збереження введених даних у `localStorage` та автоматичне їхнє заповнення після перезавантаження сторінки.

## 🚀 Функціонал

### 📷 Галерея зображень
- Динамічна генерація галереї на основі масиву даних.
- Відкриття зображень у модальному вікні (`SimpleLightbox`).
- Відображення підписів із затримкою у 250 мс.
- Адаптивна верстка.

### ✉️ Форма зворотного зв'язку
- Введені дані зберігаються у `localStorage`.
- При перезавантаженні сторінки форма заповнюється збереженими даними.
- Перевірка на заповненість полів перед відправкою.
- Виведення даних у консоль і очищення форми після успішного відправлення.

## 📦 Використані технології

- HTML, CSS
- JavaScript (ES6+)
- `SimpleLightbox` (для галереї)
- `localStorage` (для збереження даних форми)
- Prettier (для форматування коду)

## 🔧 Як запустити

1. **Склонуйте репозиторій:**
   ```sh
   git clone <URL-репозиторію>
   ```
2. **Перейдіть у папку проєкту:**
   ```sh
   cd goit-advancedjs-hw-01-main
   ```
3. **Встановіть залежності:**
   ```sh
   npm install
   ```
4. **Запустіть локальний сервер:**
   ```sh
   npm start
   ```
