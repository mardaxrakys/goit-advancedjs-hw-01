# 📸 Галерея зображень на SimpleLightbox

Цей проєкт реалізує галерею зображень з використанням бібліотеки **SimpleLightbox**.  
Користувач може переглядати мініатюри, відкривати повнорозмірні зображення в модальному вікні, гортати їх за допомогою клавіатури та бачити підписи.

---

## 📦 Використані технології

- HTML5
- CSS3
- JavaScript (ES6+)
- [SimpleLightbox](https://simplelightbox.com/) — бібліотека для модального перегляду зображень
